# markdown lint

language: [🇯🇵](README.md)   [🇺🇸](docs/README_en.md)

Markdownを静的解析し、自動的に整形をします。

## 使い方

### そのまま使用

```shell
# インストール
pip install git+https://github.com/yuto51942/markdown-lint

# 起動
md-lint
```

### Pythonライブラリとして使用

```shell
# インストール
pip install git+https://github.com/yuto51942/markdown-lint
```
